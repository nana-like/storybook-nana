# 버튼

## 형태

- appearance = `fill` | `outline` | `text` | `pure`
- `pure`는 버튼 자체의 스타일링 없이 버튼 태그만 필요할 때 사용합니다.

    <MyButton appearance="pure">pure 버튼</MyButton>
  </Story>
</Canvas>

## 색상

- color = `primary` | `secondary` | `tertiary`

<Canvas>
